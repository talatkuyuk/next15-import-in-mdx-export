### 1. Image syntax in markdown / mdx (in `public` directory)

Normally, we put the images in **`public`** directory, and use **root-relative links** (`/pathname-to-image`)
for referring it. Pay attention to that the images should be in the public directory.

| format    | syntax                                        | proof                                                                                   |
| :-------- | :-------------------------------------------- | --------------------------------------------------------------------------------------- |
| markdown  | `![alt](/public-image.png)`                   | ![alt](/public-image.png "public image > 60x60")                                        |
| html      | `<img alt="alt" src="/public-image.png" />`   | <img alt="alt" src="/public-image.png" title="public image > 60x60" />                  |
| component | `<Image alt="alt" src="/public-image.png" />` | <Image alt="alt" src="/public-image.png" width={60} height={60} title="public image" /> |

### 2. Image syntax in markdown / mdx (in a custom directory)

We should create **a custom script** to copy the blog assets
which are placed alongside with the articles into output directory during export.

Thanks to that `script`, the **`data`** directory containing the blog assets will be accessible.
To verify this, let's display the images from the **`data`** directory.

| format    | syntax                                                                                                        | proof                                                                                                                              |
| :-------- | :------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| markdown  | `![alt](/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png)`                   | ![alt](/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png ">60x60")                                 |
| html      | `<img alt="alt" src="/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png" />`   | <img alt="alt" src="/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png" title=">60x60" />           |
| component | `<Image alt="alt" src="/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png" />` | <Image alt="alt" src="/data/articles-strategy-scoped-content/article-1/article-assets/article-image.png" width={60} height={60} /> |
