### Import A Component with `.jsx` extension

It throws error `Unknown file extension ".jsx" for /Users/.../mdxComponents/Bar.jsx`,
since imported components should be pure javascript not jsx.

The **relative link** is relative to the module which **`MDXRemote`** called.

import Bar from "../../../mdxComponents/Bar.jsx";

<Bar />;
